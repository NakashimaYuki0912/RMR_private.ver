A主菜单 → 再进初始遗物（推荐）

  • 正常游玩 → 关闭主菜单 → 初始遗物选择 → 章节路线
  • 挑战解放战 → 楼层选择 → 打完回主菜单
  • 玩法介绍 → 说明页 → 回主菜单
  • 退出 → 回图书馆

遗物选择里不再出现解放战按钮。

B主菜单直接开跑（跳过独立遗物事件）

  • 点「正常游玩」后不再弹初始遗物，或把遗物并进后续步骤
  • 直接进入第一章节点选择

改动更大，会动到开局节奏。
