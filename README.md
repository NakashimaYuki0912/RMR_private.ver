## 安装方法

1. 下载本页面 Release 附件中的 `RougelikeModReborn_private-version_vxxxx.zip`。
2. 解压 zip 文件。
3. 打开《Library Of Ruina》的 Mods 目录：

   `...\Steam\steamapps\common\Library Of Ruina\LibraryOfRuina_Data\Mods`

4. 在 `Mods` 目录中新建文件夹，例如：

   `RougelikeModReborn`

5. 将解压后的所有文件放入该文件夹中，最终结构应类似：

   `LibraryOfRuina_Data\Mods\RougelikeModReborn\RogueLike Mod Reborn.dll`

   `LibraryOfRuina_Data\Mods\RougelikeModReborn\StageModInfo.xml`

   `LibraryOfRuina_Data\Mods\RougelikeModReborn\AddData`

   `LibraryOfRuina_Data\Mods\RougelikeModReborn\SpecialStaticInfo`

6. 如果安装过旧版本，请先删除旧的 mod 文件夹，再安装新版，不建议直接混合覆盖。

## 注意事项

- 不要让 zip 解压后多套一层文件夹。
- `RogueLike Mod Reborn.dll` 和 `StageModInfo.xml` 必须位于 mod 文件夹第一层。
- 更新版本时建议删除旧版文件夹后重新放入新版。
